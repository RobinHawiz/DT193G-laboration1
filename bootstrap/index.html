Tailwind

Bootstrap är ett öppet källkodsramverk för CSS. Jämfört med utility-first ramverket Tailwind fokuserar Bootstrap mer på färdiga komponentklasser (till exempel knappar, tabeller, formulär och navbars). Bootstrap erbjuder också många utility-klasser och sedan version 5 kan de utökas via Utility API. Utöver CSS innehåller Bootstrap ett JavaScript-bibliotek som ger interaktivitet åt komponenterna, till exempel tooltips, dropdowns och modaler. I mitt formulär använder jag en tooltip på knappen Skicka, vilket kräver att tooltips skapas via Bootstrap JS.

En fördel med Bootstrap är att ramverket avsevärt underlättar frontendutveckling tack vare färdigstylade komponenter. Behöver man en design som är tillräckligt bra och vill lägga mer tid på funktionalitet är ramverket utmärkt. Har man däremot mer tid för frontend och vill skapa en mer skräddarsydd design kan de färdigstylade komponenterna bli en nackdel, eftersom de kräver mer arbete att anpassa.

Min erfarenhet av Bootstrap är överlag mycket positiv. Att jag först lärde mig Tailwind CSS hjälpte, eftersom båda ramverken stylar HTML-element via klassnamn. När jag jämför tiden det tog att styla en sida med Tailwind med den här sidan som stylades med Bootstrap gick utvecklingen betydligt snabbare med Bootstrap, främst tack vare de färdigstylade komponenterna.
